TextTools API

A collection of AI-powered text processing tools.

Available Tools

  • Categorize – Classify text into predefined categories.
  • Extract Keywords – Extract key terms and phrases.
  • Extract Entities – Identify named entities (people, places, etc.).
  • Is Question – Detect if a text is phrased as a question.
  • Generate Questions – Create questions from a text or subject.
  • Merge Questions – Combine multiple questions into one.
  • Augment – Rewrite text in different styles (positive, negative, etc.).
  • Summarize – Generate a concise summary.
  • Translate – Translate text between languages.
  • Propositionize – Break text into atomic propositions (experimental).
  • Is Fact – Verify a statement against a source text (experimental).

Powered by FastAPI and OpenAI.
